# 操作分支相关命令

- 查看本地分支： git branch
- 查看远程分支： git branch -r
- 查看所有分支： git branch -a
- 创建本地分支： git branch [name]
- 切换本地分支： git checkout [name]
- 新建并切换分支： git checkout -b [name]
- 删除分支： git branch -d [name]
- 合并分支： git merge [name]
- _创建远程分支(本地分支 push 到远程)： git push origin [name]_

# 远程仓库相关命令

- 查看远程仓库： git remote -v
- 删除远程仓库： git remote rm [name]
